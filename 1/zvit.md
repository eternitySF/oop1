## Короткі теоретичні відомості
Git Bash - це програма для середовищ Microsoft Windows, яка емулює командну стрічку Git. Ця програма може напряму взаємодіяти з Git Hub і вносити змінні. Програма легко переміщується по папкам комп'ютера і робить користування зручним.
Git Hub - це найпопулярніше віддалене сховище для git-репозиторіїв, яке дає можливість легко комунікувати між програмістами та обмінюватись даними.
## Хід роботи 
1. Завантажив Visual Studio Code для Windows 10,11 з Google Chrome. Встановив. Відкрив програму та встановив розширення Python.
2. Написав базову [програму](./script.py) на мові Python 

Висновок:
1) Я користувався GitHub, дізнався про Git Bash та вперше "запушив" програму в середовище.
2) Дізнався команди навігації в програмі Git Bash, закріпив для себе нові команди роботи з середовищем GitHub.
3) Знаю як робити нові репозиторії в GitHub  
4) Дізнався як ще раз "запушити" та додати оновлені файли в мій репозиторій GitHub
5) Ознайомився з новою схованою папкою .git та дізнався для чого вона.
